Expected CSV format

Required columns (header names are matched loosely, case-insensitive):

  • Market (or Event / Question) — the market name
  • Entry (or Buy Price / Price) — entry price, 0–1
  • Exit (or Sell Price / Close Price) — exit price, 0–1
  • Outcome (or Result / Status) — Yes / No / Draw
  • Profit (or P&L / PnL / Profit/Loss) — in USD

Optional: a Date column unlocks the timeline slide. Rows with empty or $0 profit are skipped. "$" signs, "%" signs, commas, and (parentheses) for negatives are handled automatically.
